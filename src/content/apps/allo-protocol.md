Allo Protocol is an open-source protocol for capital allocation built by Gitcoin. It provides modular building blocks (strategies) that developers can use to create any type of funding mechanism. Allo V2 powers Gitcoin Grants Stack and enables custom allocation strategies for quadratic funding, direct grants, and more.
